6. Points Calculation Logic Rules
Event Points Awarded Condition
Win a match +2 pts match_results.status = approved, player wins majority of sets
Lose a match +1 pt match_results.status = approved, player loses majority of sets
Early-bird bonus +2 pts Player completes 3+ approved matches before tournament midpoint date
Standings sort order Points → Wins → Losses → Sets Won Descending points first, tiebreakers applied in order
